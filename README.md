# fengnovoblog
express,ejs,nodejs,blog  
一步步按照 https://maninboat.gitbooks.io/n-blog/content/ 写的博客系统

## 环境要求  
必须安装node,mongodb,express

## 开发环境  
```
npm install  
npm start  
```

## 测试环境
```
npm install  
npm test  
```


## 部署生产环境
```
npm install  
npm run build  
```

## 测试test运行  
```  
npm install  
npm test  
```

### pm2运行  
```  
npm install  
npm production  
```
